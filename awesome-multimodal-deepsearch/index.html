READING NOTE

如何读这张地图

先比较问题定义。一项工作可能评估搜索流程，另一项评估网页浏览、视觉 query 修复或通用工具协作；它们的准确率通常不可直接横比。

再检查视觉是否不可替代。图像是初始提示、最终证据，还是会反复决定下一跳搜索？这三种设置对应不同难度。

最后看过程信号。只有最终答案 reward 容易掩盖错误轨迹；checklist、sub-goals、entity sequence 和 reasoning graph 提供不同粒度的监督。
